SecureMe

Java | Android SDK | Android Studio

SecureMe is a simple Android app targeted at platforms 2.3.3+ (which covers ~99% of all Android platforms in use) that automaically helps users improve security settings on their Android devices. SecureMe checks devices for common security settings like securing a lock screen with a password, PIN, or pattern lock; enabling encryption; enabling remote acess with Android Device Manager; and other security-related configurations. Instead of users having to look up how to secure their devices, SecureMe tells users at a quick glance which security settings they have configured well (either manually or by default) and which settings they should take a look at.

bwReport

Python | Cisco NetFlow

bwReport is an internal command-line interface tool for quick, simple, and precise bandwidth usage analysis. It is useful for gathering network usage statistics over both Ethernet and RADIUS Accounting (RFC 2866) Wi-Fi sessions (hosts behind NAT).

Class projects UC Berkeley EECS

CS 168: Networking Architecture and Protocols

Bears Transport Protocol (BTP)

Python

Bears-TP is a simple reliable transport protocol that provides in-order, reliable delivery of UDP datagrams in the face of packet loss, delay, corruption, duplication, and re-ordering. BTP uses fast retransmit and selective acknowledgments to optimize performance.

Distance-vector router emulator

Python

The DV router emulator mimics the behavior of routers in a network of distance-vector routers. Each router is an instance of a single DV router class and must correctly forward packets throughout the network in the face of new links, broken links, routing loops, and no central routing authority.

CS 161: Computer Security

Secure PNG/JPEG parser

C

The secure PNG/JPEG parser extracts data from PNGs and JPEGs in a safe and secure manner such that attackers cannot inject malicious input a PNG/JPEG as an exploit vector for attacks.

DNS spoofer

Python | Scapy

The DNS spoofer sends packets to hosts making DNS requests in order to impersonate the DNS server and launch a DNS spoofing attack, allowing us (the attacker) to redirect unsuspecting users to pages of our choosing.

CS 61C: Computer Architecture (Machine Structures)

Word co-occurrence analyzer

Java | Apache Hadoop MapReduce | Amazon EC2 cloud computing

The project answers the question: given a word, what other words are statistically associated with it? For example, what words and concepts go with love or death? Co-occurrence measures how often two words appear together in documents; given a target word, we can find words in a body of text that are closely related to it by ranking each unique word in a document by its co-occurrence rate with the target word.

MIPS instruction set emulator

C

This project parses instructions in the MIPS ISA to simulate a processor operating on the MIPS instruction set. Sample instructions include bit shifting, reading/writing in memory, arithmetic operations, conditional statements, instruction execution logic jumps, and system calls.

Image processing and optimization

C | OpenMP

The image processor performs 2D convolution on a given image with a given kernel to achieve image "filter" effects such as sharpening, blurring, and inverting. The project focuses on optimization of the 2D convolution process using caches, multi-threading, and compiler optimization. Specific optimization techniques include SSE instructions for batch processing, register blocking, loop unrolling, matrix padding, cache blocking, and OpenMP for quick and correct multi-threading and parallel processing.

Processor design

Logisim

The final project was a simulated 16-bit processor in Logisim complete with a specified ISA including arithmetic operations, reading/writing to registers and memory, and instruction logic such as jumping and branching.

CS 61B: Data Structures

Sharks and Fish (simulation)

Java

Sharks and Fish simulates an ocean containing sharks and fish with created and destroyed given a set of rules, similar to Conway's Game of Life. The ocean was compressed using run-length encoding.

Network (the game)

Java

Network is a 2-player game on an 8-by-8 board. A player can win by connecting six or more chips orthogonally or diagonally from the player's edge of the board to the opposite edge of the board. Players can play against simple artficial intelligence logic that uses alpha-beta pruning and the minimax algorithm to determine the optimal moves to take.

Weighted undirected graphs and MSTs

Java

This final project combines a variety of data structures and algorithms to implement a well-encapsulated abstract data type to represent a weighted undirected graph. The second part of the project implements the greedy Kruskal's algorithm to find a minimum spanning tree in the weighted undirected graph.

CS 61A: Structure and Interpretation of Computer Programs

Hog (the game)

Python

Hog is a dice game where the objective is to reach a score of 100. Among several rules, players can keep rolling to add to a running total of points to add to their score, but rolling a 1 will result in only a single point added for that turn. This text-based game implements several different strategies to achieve victory using probability and other heuristics.

Twitter trends is a simple data visualization project that collects public tweets tagged with geographic locations, assigns a sentiment to each tweet based on all words in the tweet, aggregates tweets by closest geographic centers, and colors each US state according to the aggregate sentiment of its tweets.
